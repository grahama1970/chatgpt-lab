# Artwork provenance

`chatgpt-lab-header.webp` was generated specifically for this repository on 2026-06-25 using OpenAI image generation and then exported as a 2:1 README banner.

## Concept

- photorealistic retro-futurist scientific laboratory;
- human brain displayed as a non-gory museum specimen beneath a glass bell jar;
- substantial dark walnut plinth;
- etched brass plaque reading `CHATGPT-LAB`;
- supporting visual language: observation, reasoning, evidence, verification, and improvement.

## Intended meaning

The brain represents reasoning. The glass makes the reasoning inspectable rather than mystical. The walnut plinth and brass plaque turn the system into a durable laboratory instrument. The archival setting connects iteration to disciplined experimentation and recorded evidence.

The repository copy is an optimized 600 × 300 WebP so the README loads quickly. The higher-resolution source remains in the originating ChatGPT-Lab conversation and can be re-exported when a larger asset is required.

Do not replace the image with a generic AI gradient, glowing circuit brain, or unsupported dashboard artwork without recording a new design decision in `sources/control-plane/DECISIONS.md`.
