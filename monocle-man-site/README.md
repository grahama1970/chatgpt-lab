# Monocle Man — React benchmark fixture

This folder is the bounded Monocle Man benchmark surface for ChatGPT-Lab. It lives in `grahama1970/chatgpt-lab` on `main` and is changed through isolated ChatGPT-authored branches and pull requests.

## Scope

The vague product ask is intentionally simple: turn the Monocle Man film into a polished landing page. The experiment is whether ChatGPT can convert that ask into a deterministic React implementation with evidence instead of vague prose.

This slice implements the landing page only. Slides, chat UI, login, databases, personalization, and broader PhatGPT/controller architecture are out of scope for this benchmark round.

## Local commands

```bash
npm install
npm run build
npm run test:benchmark
```

The benchmark emits `benchmark-evidence/` with interactions, console/network reports, accessibility output, screenshots, image status, rendered source, artifact manifest, and evidence-derived `verdict.json`.

## GitHub Pages delivery

`.github/workflows/monocle-man-pages.yml` builds `monocle-man-site` and deploys the generated `dist/` directory to GitHub Pages on pushes to `main`.

GitHub Pages is static hosting. That is compatible with this React/Vite app because the app has no server-side routes and no React Router deep links; it is delivered as static HTML, CSS, and JavaScript generated by `npm run build`.

The Vite config uses `base: "./"` so built asset URLs are relative to the deployed page location.

## Historical Netlify config

`netlify.toml` remains in the fixture for compatibility with earlier review infrastructure, but the current GitHub-first delivery gate is GitHub Pages plus benchmark CI evidence.
