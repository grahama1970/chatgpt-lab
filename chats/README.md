# Chats

This directory stores exported or summarized ChatGPT-Lab conversations.

Use these files for intent, decision history, and project context. Do not treat a chat export as proof that code ran, a deployment succeeded, or a visual result was inspected. Execution proof belongs in `artifacts/` and structured run history belongs in `iterations/`.

Current chat exports:

- `monocle-man-website-creation.md`: initial conversation about packaging the Monocle Man website work into this project repository.
